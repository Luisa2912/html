Definiciones de terminos informaticos

HTML
HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje base para la creación de páginas web.
CSS
Cascading Style Sheets (Hojas de Estilo en Cascada). Se utiliza para definir el estilo de presentación de las páginas web.
JavaScript
Lenguaje de programación interpretado que se utiliza para añadir interactividad y dinamismo a las páginas web.
API
Application Programming Interface (Interfaz de Programación de Aplicaciones). Conjunto de definiciones y protocolos que permiten a un programa acceder a los servicios de otro.
Base de datos
Almacén organizado de información digital. Se utiliza para almacenar, organizar y recuperar datos.
Servidor
Equipo informático que proporciona servicios a otros equipos en una red. Puede ser un servidor web, un servidor de correo electrónico, etc.
Cliente
Equipo informático que solicita servicios a un servidor. Puede ser un navegador web, un cliente de correo electrónico, etc.
Red informática
Conjunto de equipos informáticos conectados entre sí para compartir recursos e información.
Internet
Red global de redes informáticas que permite la comunicación y el intercambio de información entre personas y dispositivos de todo el mundo.
Cloud computing
Modelo de computadion en la nube. Permite acceder a servicios informaticos (servidores, almacenamientp, base de datos, etc) a traves de internet.
